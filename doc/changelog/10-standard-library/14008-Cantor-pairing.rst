  ``Cantor.v`` containing the Cantor pairing function and its inverse.
  ``Cantor.to_nat : nat * nat -> nat`` and ``Cantor.of_nat : nat -> nat * nat``
  are the respective bijections between ``nat * nat`` and ``nat``.
